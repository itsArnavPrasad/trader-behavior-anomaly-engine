# AnomX – Trader Behavior Anomaly Detection Engine

## Our Mission
Build an intelligent real-time system that detects suspicious trading and user behavior to create a safer financial ecosystem.

---

## About the Project

AnomX is a real-time anomaly detection system for trading platforms. It analyzes user activities (logins, transactions, trades, sessions, etc.) to detect suspicious patterns and generate actionable alerts for compliance teams.

This project is inspired by real-world industry challenges and is designed to teach you end-to-end AI system building in a beginner-friendly way.

---

## Key Objectives

- Detect anomalous users using unsupervised learning
- Process streaming events in real-time or near real-time
- Provide explainable anomaly scores
- Generate human-readable alerts
- Build a production-style ML system from scratch
